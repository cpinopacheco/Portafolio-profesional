Sorteo de rifa

La idea de este proyecto surge a raíz de una situación en particular. Cuando observe a una persona sorteando una rifa de manera online y buscaba en internet un generador de número aleatorios, pero muchos de estos programas eran muy simples y poco estilosos. Entonces en ese momento se me ocurrió crear un programa diferente, algo que no solo cumpliera con las necesidades principales de un sorteo de rifa, sino también, que fuera llamativo e intuitivo con el usuario.

El programa le permite al usuario ingresar la cantidad de números que participaran de la rifa, además de los premios que se sortearan. Renderizando esta información en pantalla para que todos los participantes esten al tanto de la información. También permite obtener números ganadores como perdedores de forma aleatoria, que se van visualizando en diferentes secciones en la aplicación. El programa también cuenta con popup informativos que ayudan al usuario con la interacción. Todo esto se realiza en una interfaz con un lindo diseño y animaciones para brindar una mejor experiencia al usuario.

Pokedex

Este proyecto permite al usuario buscar información acerca de las famosas criaturas conocidas como "Pokémon" de una manera diferente, atractiva y divertida. El desarrollo de este proyecto surge para poner en práctica mis conocimientos en Html, Css, JavasScript. El objetivo principal es interactuar, a través de AJAX, con la API externa de Pokémon, para obtener la información que esta nos brinda y poder asi renderizarla en pantalla.

Para llegar más allá, quise imitar las caracteristicas de la pokedex original, para conseguirlo, agregue funcionalidades extras como un botón de encendido y apagado, efectos de sonido, animaciones y control de errores para mejorar la experiencia. Finalmente busque la manera que la pokedex no solo mostrara la información por pantalla, sino tambíen, que hablara. Eso lo conseguí a través de la interacción con la web API del navegador conocida como 'SpeechSynthesis'.

Películas Populares

Este proyecto fue creado para seguir poniendo en practica mis conocimientos, en esta ocasión quise realizar un sitio web responsivo, que tuviera un buscador y paginación. Para lograr esto, trabajé con el consumo de la API de The Movie DataBase (TMDB).

Para entrar en contexto, (TMDB) es una base de datos de películas y TV creada por su comunidad. Cada pieza de dato ha sido agregada por la comunidad desde el año 2008.

A través de este proyecto se puede obtener y consultar las películas más populares en la actualidad, obteniendo información relevante de cada una de ellas. Toda la funcionalidad del sitio se ejecuta de manera asíncrona en un solo archivo HTML.

Curriculum digital

Estaba en la búsqueda de tener algo interesarante para maquetar y así poder practicar mis conocimientos en CSS Grid y responsive web desing, ahí surgió la idea de convetir mi curriculum en formato PDF, a un curriculum completamente digital. Esto me serviría como ejercicio para trabajar con un sistema de cuadrícula y realizar un sitio completamente responsivo.

El número de teléfono y la dirección de email que se encuentran en la parte superior, en la cabecera del curriculum, están enlazados para brindar un contacto directo. Debajo de la descipción personal, se encuentra un enlace que que redirigire directo a mi portafolio porfesional. Además, agregue un botón que le permite al usuario cambiar la tonalidad completa del sitio a un modo oscuro o luminoso con la finalidad de mejorar la experiencia del usuario. Guardando esta configuración en el almacenamiento local del navegador (local storage), de esta manera, la próxima vez que el usuario vuelva a visitar al sitio, cargará la última configuración.

Jumping

Siempre me ha interesado el desarrollo de videojuegos, es por este motivo que realice un curso básico de unity para adentrarme a este maravilloso mundo, es completamente divertido crear algo desde 0 y después ver el resultado final. Elegir los sprites(personajes), darles movimiento, agregar efectos de música y sonido, entregar una experiencia jugable por mas simple que sea es genial.

Jumping es un videojuego estilo pixelar en 2D, y consiste en ir saltando enemigos para ir acumulando puntos. Entre más avanzas va subiendo la dificultad lo que provoca que los enemigos avancen más rápido, pierdes en el juego cuando colisionas con alguno de los enemigos.

Jumping lo desarrolle como proyecto final del curso, en el año 2018. Cuando estaba cursando mi carrera como programador, esto me sirvió para ir entendiendo y aprender más sobre programación en general.
